Clara Parabricks is a suite of genomics software tools that aim to use GPU acceleration to speedup commonly used genomics tools like BWA. NVIDIA claims speedups of up to 80x compared to their serial CPU equivalents. Historically, the tool has required a paid license to use, but since GTC 2022, they have made it free for academic purposes, meaning WEHI researchers can use it free-of-charge!

ITS have setup Clara Parabricks centrally via the Environment Modules system to make using the containers simpler. Access it by

module load clara-parabricks/4.0.1-1

See our January 2023 RCP Newsletter for some basic speed comparisons with CPU equivalents on Milton CPUs/GPUs.

Usage

Note that for the tool to work (including using the --help option), you must run the module on a Milton node with GPU(s). You will receive an error otherwise.

The software suite is published as a collection of Docker containers, and requires familiarity with Apptainer to use them on Milton. However, the central installation we have at Milton is simplified, so the apptainer run command and options can be ommitted. For example, to run the germline pipeline tool:

module load clara-parabricks/4.0.1-1\ngermline \\\n    --ref in_dir/ref.fasta \\\n    --in-fq in_dir/1.fq in_dir/2.fq \\\n    --knownSites in_dir/known-sites.vcf.gz \\\n    --out-bam out_dir/mark_dups.bam\n    --out-variants out_dir/variants.vcf\n    --out-recal-file out_dir/recal.txt\n

The Clara Parabricks module has every tool in the software suite available. See the Clara Parabricks documentation for each tools' usage.
