DIA-NN

DIA-NN is a universal software for data-independent acquisition (DIA) proteomics data processing by Demichev, Ralser, and Lilley labs. In 2018, DIA-NN opened a new chapter in proteomics, introducing a number of algorithms that enabled reliable, robust, and quantitatively accurate large-scale experiments using high-throughput methods.

Read our DiaNN step-by-step guide for more information.

In the next examples, please change the input files (--f), the library files (--lib), output files (--out) and output library files (--out-lib) according to your data.

The output directory (--out) should be created before the run.
DiaNN will be unable to save output files if this directory does not exist.

Remember to create this directory before running.

Example 1: Library-based search job submission script 

#!/bin/bash\n  \n#SBATCH -p regular\n#SBATCH --job-name=DiaNN_library_based\n#SBATCH --time=48:00:00\n#SBATCH --cpus-per-task=45\n#SBATCH --mem-per-cpu=5G\n\nmodule use /stornext/System/data/modulefiles/sysbio\nmodule load DiaNN/1.8\n\ndiann-1.8 \\\n    --f input_file1.d \\\n    --f input_file2.d \\\n    --lib lib_file \\ \n    --threads 44 --verbose 1 --out output_file \\\n    --qvalue 0.01 --matrices  --out-lib output_lib_file \\\n    --gen-spec-lib --predictor \\\n    --fasta Databases_file.fasta \\\n    --met-excision --cut K*,R* --mass-acc 10 --mass-acc-ms1 10.0 \\\n    --use-quant --reanalyse --smart-profiling --peak-center \\\n    --no-ifs-removal\n


Example 2: Library-free search job submission script

#!/bin/bash\n  \n#SBATCH -p regular\n#SBATCH --job-name=DiaNN_library_based\n#SBATCH --time=48:00:00\n#SBATCH --cpus-per-task=45\n#SBATCH --mem-per-cpu=5G\n\nmodule use /stornext/System/data/modulefiles/sysbio\nmodule load DiaNN/1.8\n\ndiann-1.8 \\\n    --f input_file1.d \\\n    --f input_file2.d \\\n    --lib "" \\\n    --threads 44 --verbose 1 \\\n    --out output_file \\\n    --qvalue 0.01 --matrices  --out-lib output_lib_file \\\n    --gen-spec-lib --predictor \\\n    --fasta Databases_file.fasta \\\n    --fasta-search --min-fr-mz 200 --max-fr-mz 1800 --met-excision --cut 'K*,R*' \\\n    --missed-cleavages 1 --min-pep-len 7 --max-pep-len 30 \\\n    --min-pr-mz 300 --max-pr-mz 1800 --min-pr-charge 1 \\\n    --max-pr-charge 4 --mass-acc 10 --mass-acc-ms1 10.0 \\\n    --use-quant --reanalyse --smart-profiling --peak-center \\\n    --no-ifs-removal
