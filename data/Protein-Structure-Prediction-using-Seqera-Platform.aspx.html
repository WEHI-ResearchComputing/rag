AlphaFold using Nextflow

RCP has developed a nextflow pipeline to run Alphafold on Milton. The code is found in our GitHub repo. 

 

The pipeline can be run from the command line, described in the README.

 

The pipeline is also available on the Seqera platform (aka Nextflow Tower).

Getting access to the WEHI workspace on Seqera Platform

Seqera Platform is the centralized command post for the management of Nextflow data pipelines. It offers monitoring, logging, and observability for distributed workflows and simplifies the deployment of pipelines on any cloud, cluster, or laptop.

Users can launch pre-configured pipelines with ease. Workflow developers can publish pipelines to shared workspaces and administrators can set up and manage the infrastructure required to run data analysis at scale.

 

To get access to WEHI's workspaces, please follow the instructions here and request access to WEHI workspace.

Running Alphafold Pipeline in Seqera Platform

Before using the pipeline, you need to prepare your fasta files and get the full path to the directory that includes all sequences you want to fold.

Further details here.

After gaining access to WEHI workspace and logging in, choose WEHI workspace, and from the list of pipelines in the Launchpad choose ProteinStructurePrediction, then press Launch.

Then fill the form with the pipeline parameters. Description and examples for each fielsd are displayed below the field. Once all is filled, press Launch.

Outputdir and inputdir should be a full absolute path.

You can monitor the status of your running pipelines from the Runs tab. 

And you can further check the status by clicking on your pipeline and you will get a report on how many tasks has finished and how many still being processed. If the pipeline has completed successfully, you will see a report on resource utilisation.
