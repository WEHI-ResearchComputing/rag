Glossary


TermMeaning
MiltonWEHI's HPC or private research computing cloud.
bashThe most common version of the Unix shell.
coreThe component of a processor ship that executes instructions. Desktop chips typically provide 2-6 cores, the chips on the HPC provide 14 cores.
CPUCentral Processing Unit - executes the instructions that make up software. Used in a couple of ways:
  1. When referring to hardware it is a physical chip, e.g. an Intel
  2. At the operating systems and programming level, it refers to a thread of execution. Modern physical chips provide many threads of execution.
hyperthreadEach core is capable of interweaving 2 threads of execution and thus appearing to the software as two CPUS (in the second sense of the term above).
LinuxThe most common version of Unix and the one used at WEHI.
nodeA ‘computer’, i.e. a quantity of memory, a number of physical CPU chips (typically 1-4), hard drives, network and other connections.
processorWhen specifying resources to the queueing system a processor refers to a hyperthread. Thus, if you request two processors you will get one core.
scriptA file with a series of commands in a language such as bash, Python, R, Perl, etc, to perform a repetitive task.
shellA Unix interface where the user types commands and text output is printed on the screen.
SLURMWEHI's batch system for running jobs.
UnixA type of operating system (like Windows or Mac OSX).
vc7-sharedA specific shared instance of Linux available to all WEHI staff and students.
virtual machine (VM)A node implemented in software (by VMWare). To users it looks like a real node. All systems at WEHI are VMs.
VMWareData centre software for running Milton.
vRAvRealize Automation - a component of VMWare that builds virtual machines.
