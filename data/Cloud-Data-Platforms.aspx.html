Multi-Cloud Project

As research datasets grow in size and complexity, security concerns grow consequently, especially when sensitive data is involved. Some institutes and organisations have started using cloud providers to build secure platforms where sensitive and/or clinical data reside and can be processed. An example is the UK BioBank and AMP Parkinson's Disease which uses Terra.bio and DNAnexus for data analysis. Both make use of public cloud providers. In addition to that, researchers occasionally need more powerful resources or a huge number of resources that are not available in their internal HPCs.

Research Computing is running a multi-cloud pilot to focus on answering some of the key questions around understanding the best options for researchers and also increasing oversight of cloud utilisation. We understand that there is no one-solution-fits-all, and that is why this pilot includes multiple projects and covers multiple cloud providers

The pilot will attempt to identify key governance needs for researchers and professional staff for current use cases and beyond, and the focus will be on accessing compute resources and storage for research workflows.   

If you are currently using any of the cloud providers, please get in contact with Research Computing, to discuss how including you in the WEHI cloud organisation structure will benefit your projects and provide access to more discounted services and offers.

So far we are providing support to Bahlo lab on their research project that uses data from AMP-PD and processes it on Terra.bio. We are also supporting the Colonial Foundation Healthy Aging Center in building a secure platform using AWS Service Workbench.
