The Milton operating system will be upgraded in the outage occurring from Thursday 15 to Monday 19 August 2024. After the outage, the following changes will be noted.

 

  1. It will no longer be possible to write to StorNext from compute nodes. Home directories are excluded, but any other location starting with /stornext will be impacted. OnDemand apps such as RStudio, Jupyter, Milton virtual desktops, and jobs submitted using sbatch, srun, salloc will be affected. For more information, click here.
  2. System libraries will be upgraded, potentially breaking certain software. This may impact some R and Python packages and compiled software, but most of these should have no issues. Any errors will be resolved by re-installing the package. For more information, click here.
  3. The /wehi and /wehisan path shortcuts will stop working Replace these paths in your scripts with the real path starting with /stornext. For more information, click here.

 

If you have any concerns, comments or inquiries please email research.computing@wehi.edu.au.
