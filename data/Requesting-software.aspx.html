Requesting new software

The nature of research is such that new software is continually being written to support new instruments and new analyses.

The software you need may already be available, so check our modules guide to determine if the software is available in the module system. Many software packages can be easily installed by users, see our guide for installing software that follows common conventions.

If the software is not available or you are having difficulty installing it, or are just seeking advice please submit a request to support@wehi.edu.au. Please provide:

  • a link to the software
  • where you need it installed
  • any problems you have encountered
  • indicate if the software will definitely be used by a number of people in the long term.

If it is beyond the scope of what users can be expected to install and maintain, Research Computing (RC) will install it. If it is not, RC will help you to install it yourself in your home area.

Please note:

Whilst some software may be easy to install on your own system, installing to a shared system with multiple versions of software (i.e. the Milton HPC cluster) can be complex and require manual installation of many software dependencies.

For this reason, it may not always be possible to install the requested software, or installing such software can take a long time.
