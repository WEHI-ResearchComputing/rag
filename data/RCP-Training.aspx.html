  • Nix and BioNix, 19 October

    • Presented By: Justin Bedő
    • Duration: 240 minutes over two days
    • Prerequisites:
      • Basics of Linux command line
      • Basics of bash scripting 
      • Install Nix on your laptop from managed software center, or by sending a helpdesk ticket
    • Course content:
      • Why reproducibility is difficult?
      • How Nix/BioNix resolves the issues using concepts from functional programming
      • Introduction to the Nix configuration language
      • Using Nix as a package manager and build system
      • BioNix as a workflow specification layer and library of bioinformatics tools
      • Using BioNix to execute Bioinformatics workflows
    • Support Materials: https://github.com/WEHI-ResearchComputing/BioNix-Training
    • ​​​​​​​​​​​​​​Recording: day 1, day 2​​​​​​​​​​​​​​
