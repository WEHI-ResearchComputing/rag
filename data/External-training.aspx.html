Recommended learning material and resources

This is a list of training and tutorial material that introduces the basic skills needed for the efficient use of scientific computing resources.

If you know of other material that you think would be helpful to others please let us know. Similarly, if this material is unhelpful, outdated, or links are broken, please email the Research Computing (RC) team.

Classroom training

Melbourne Bioinformatics offers training, but access may require an honorary or co-appointment at Melbourne University in some cases. Coursera has a lot of material for programming and data science. It seems to be free if you don't require a certificate. Similarly, DataCamp allows some free access to training material.

More Resources

UNIX concepts, the UNIX command line and shell scripting

The primary way of interacting with Unix systems (which are almost universally Linux these days) is via the command line. Mac OSX also has an excellent command-line interface available through the Terminal program. Once comfortable with the command line you should learn the basics of shell scripting which will allow you to make new commands to help with your work, automate repetitive tasks and manage complex workflows.

​​​​​​UNIX editors

You will need to edit text files as part of your work. UNIX text editors originated in a world where there were no graphical user interfaces and are an arcane world unto themselves. There are some relatively easy-to-use editors:

  • joe
  • nano 

vi or vim are very popular and powerful editors but difficult to learn. Here are some introductory guides.

  • Vim adventures
  • Open Vim

In addition, vim comes with built in training. At the command line type vimtutor.

Power users aspire to Emacs:

  • www.gnu.org/software/emacs/tour/
  • www.jesshamrick.com/2012/09/10/absolute-beginners-guide-to-emacs/  

Sublime Text is a popular editor on Mac OSX and Windows PCs. It is a commercial product so you need to purchase a licence (or get your lab head to buy it for you). There is a version available for Linux but you need to set up an X server on your PC and use X-windows tunnelling when connecting to the target system (not covered here).

Programming languages

Modern science involves the exploration and visualisation of complex data. This can only be accomplished by writing your own programs. There are many languages and language ecosystems used in science and engineering but the main ones used at WEHI are Python and R.

Using cloud services

Qwiklabs provides very good guided training for Amazon Web Services and Google Cloud Platform. Each lesson is run using temporary credentials so you don’t need to pay for cloud usage during training and can work through the lesson on the live systems.

200 Qwiklab credits are available through Google on application for educational use, training, or self-teaching for academic staff, and to students for self-teaching. 5000 are available if you are training people in a course.
