Setting up your Shiny App

First, you will need to request a Shiny App hosting, to know more about the application process please visit Shiny Apps Hosting Page. Once you have been approved, you can start setting up your app.

For the Public Shiny Apps, the server name is vcpub-shiny​​​​​​​, and the full publicly accessible URL is https://shiny.wehi.edu.au/

To login 

ssh vcpub-shiny

For the Internal Shiny Apps, the server name is wehi-shiny​​​​​. And the full URL is http://wehi-shiny.hpc.wehi.edu.au/ only accessible from inside the WEHI network.

To login 

ssh wehi-shiny\n

Setting up your ShinyApps directory

Next you have to create a folder called ShinyApps in your $HOME directory (~)

mkdir ShinyApps

Then create a folder for your application, for example testapp

mkdir ShinyApps/testapp

Inside your application directory (~/ShinyApps/testapp)create your app.R. IF you have already created it on any filesystem on Milton you can use scp to copy it. You must be logged into Milton first, to run scp. For example, 

ssh slurm-login \nscp /vast/scratch/users/iskander.j/shinyapp/app.R wehi-shiny:~/ShinyApps/testapp/\n

This will copy my file app.R that resides in /vast/scratch/users/iskander.j/shinyapp/ into my ~/ShinyApps/testapp/ that is on the wehi-shiny (the Internal Shiny Server). You can change paths and server name according to your need.

Setting up R and app dependencies

On the servers, there is a Modules system.

To make your shinyapp use these modules, and to specify the version of R to use, modify your ~/.bashrc.

nano ~/.bashrc

Add the module load commands to the end of it, then Ctrl+X to exit then Y and enter to save. To reload the ~/.bashrc so that changes can be applied to your current session

source ~/.bashrc

To check that changes were applied

module list\n\n​​​​​​​Currently Loaded Modulefiles:\n  1) hdf5/1.12.1      2) R/4.1.2          3) openssl/1.1.1m

Next to install all required packages, open an interactive R session from the command line, and install the required packages.

Viewing and Sharing your app

Once you have set up your R environment and your app.R, you can view/share your app through 

  • For Public Shiny Apps, https://shiny.wehi.edu.au/<username>/<app directoryname>. For example, https://shiny.wehi.edu.au/iskander.j/testapp/
  • For Internal Shiny Apps, http://wehi-shiny.hpc.wehi.edu.au/<username>/<app directoryname>. For example, http://wehi-shiny.hpc.wehi.edu.au/iskander.j/testapp/​​​​​​​

Please check Shiny Server Hosting Page for more details about Storage, code management and URL management. And also refer to the Shiny Server Hosting Policy and Operating Procedure.
