Accessing cloud

What is the cloud?

Cloud computing is a paradigm whereby computational resources such as computers, storage and bandwidth can be rented on a pay-for-what-you-use basis

Cloud access at WEHI

There is currently no ITS budget to fund cloud projects. Nevertheless, please come and talk to us if you have a project that requires cloud access because there may be an opportunity to work with cloud vendors on proof of concept projects.

Typical use cases are:

  • Machines with unsupported networking such as public-facing web services
  • Complex and unsupported software for which images exist in the cloud
  • Unsupported operating systems or environments
  • Machine types that we don't have on-premise
  • Hardware not supported on-premise, such as specific GPU models or field-programmable gate arrays (FPGAs)
  • Creating and provisioning systems for teaching
  • Analysing data stored in clouds, directly in the cloud.
